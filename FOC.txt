Primary and Secondary Memory:
Primary memory: Yeh temporary memory hoti hai (jaise RAM) jo computer ke chalne tak data rakhti hai.
Secondary memory: Yeh permanent memory hoti hai (jaise hard drive), jisme data long time tak store hota hai.

RAM & ROM its types:
RAM (Random Access Memory): Temporary memory jo data ko temporarily store karti hai jab aap koi program chalate hain. Types: DRAM (Dynamic RAM), SRAM (Static RAM).
ROM (Read-Only Memory): Permanent memory jo booting aur basic functions ke liye data store karti hai. Types: PROM, EPROM, EEPROM.

LLL & HLL:
LLL (Low-Level Language): Yeh programming languages hain jo hardware ke saath directly interact karte hain, jaise Assembly Language.
HLL (High-Level Language): Yeh programming languages hain jo user-friendly hote hain, jaise Python, Java.

Virus: Yeh ek harmful program hota hai jo aapke computer ko damage kar sakta hai ya data chura sakta hai.

Interpreter and Assembler:
Interpreter: Yeh ek program hai jo line-by-line source code ko execute karta hai, jaise Python interpreter.
Assembler: Yeh low-level language (assembly) ko machine code mein convert karta hai.

DBMS, PPP, Word Document:
DBMS (Database Management System): Software jo data ko store aur manage karta hai, jaise MySQL.
PPP (Point-to-Point Protocol): Networking protocol jo direct connection establish karta hai do devices ke beech.
Word Document: Microsoft Word mein banaye gaye documents jo text aur images rakhte hain.


OS, System S/W, Apps S/W:
OS (Operating System): Software jo computer ke resources ko manage karta hai, jaise Windows, macOS.
System S/W (System Software): Aise software jo computer ki basic functioning ke liye zaroori hote hain, jaise OS aur drivers.
Apps S/W (Application Software): Wo software jo specific tasks ke liye use hote hain, jaise Word, Excel.




1. Computer kya hai?
Computer ek electronic device hai jo user ke diye gaye instructions ko follow karke data ko process, store aur retrieve karta hai. Yeh calculations aur tasks ko fast aur accurately perform karta hai

C.O.M.P.U.T.E.R:
Common Operating Machine Particularly Used for Technological and Educational Research.

2. Time
Computer ka time ke sath development hua hai, purane dino ke bade machines se lekar aaj ke modern computers tak.


3. Main Components
Input: Jo data computer ko diya jata hai (e.g. keyboard, mouse).
Processing Unit: Jahan computer ka kaam hota hai (CPU).
Memory: Data store karne ke liye. Temporary (RAM) aur permanent (Hard Disk).
Output: Jo result dikhata hai (e.g. monitor, printer).


4. VT, T, IC, LSI, VLSI Examples
VT (Vacuum Tube): Pehli generation ke computers. Example: ENIAC.
T (Transistor): Doosri generation ke computers. Example: IBM 1401.
IC (Integrated Circuit): Teesri generation ke computers. Example: IBM 360.
LSI (Large Scale Integration): Chauthi generation. Example: Intel 8085.
VLSI (Very Large Scale Integration): Paachvi generation ke modern computers. Example: Laptop, PC.


5. Size, Speed, Memory

i. Punch Card:
Pehle data store karne ke liye cards ka use hota tha.

ii. Sequential:
Data ko ek sequence me access kiya jata tha. Example: Magnetic Tape.

iii. Magnetic Tape/Disc:
Purane time me storage ke liye magnetic tapes aur discs ka use hota tha.

iv. Random:
Randomly aur jaldi se data access hota hai.

FD: Floppy Disk
HD: Hard Disk
CD: Compact Disc
DVD: Digital Versatile Disc
BRD: Blu-ray Disc

6. Characteristics (Khaas Baat)
Speed: Bahut fast kaam karta hai.
Accuracy: Galti nahi karta, sahi answer deta hai.
Automation: Apne aap instructions follow karta hai.
Storage: Data ko save karne ki shakti hoti hai.
Versatility: Har kaam ke liye use ho sakta hai.
Connectivity: Internet ya dusre devices se connect hota hai.

7. Limitation (Kami)
Soch nahi sakta: Sirf instructions ke basis par kaam karta hai.
Electricity zaroori: Bijli ke bina nahi chalta.
Maintenance: Time to time repair aur updates chahiye hote hain.

8. Generation (PeeDhi)
Pehli Generation: Vacuum Tubes (1940-1956).
Doosri Generation: Transistors (1956-1963).
Teesri Generation: IC (1964-1971).
Chauthi Generation: Microprocessor (1971-se ab tak).
Paachvi Generation: Artificial Intelligence (Abhi ke time ka).

9. Types

i. Size:
Microcomputer (Personal Computer, Laptop)
Minicomputer (Chhote business ke liye)
Mainframe (Bade companies ke liye)
Supercomputer (Sabse powerful, science ke kaam ke liye).

ii. Working:
Analog Computer: Continuous data (e.g., temperature).
Digital Computer: Binary numbers pe kaam karta hai (0 aur 1).
Hybrid Computer: Analog aur Digital dono ka combination.

iii. Purpose:
General Purpose (GP): General kaam ke liye (e.g. office work, study).
Special Purpose (SP): Khaas kaam ke liye (e.g. weather forecasting, medical machines).

1. Input Devices (Data dene wale tools):
Yeh wo devices hain jo computer ko data dene ke kaam aate hain:

Barcode Scanner: Dukan ya mall me barcode ko scan karke product ki detail computer me bhejta hai.
Keyboard: Typing ke liye istemal hota hai.
Mouse: Computer screen pe cursor ko control karta hai.
Mic (Microphone): Awaz record karne ke liye use hota hai.
OCR (Optical Character Recognition): Likhe hue text ya image ko digital text me badal deta hai. Example: Photo ya paper ka likha text computer pe type hota dikhana.
OMR (Optical Mark Recognition): Exam sheets jaise forms me filled circles ko scan karke result nikalta hai.


2. Processing (Data ko process karna):
Data ko process karne ke liye computer ke andar kuch main parts hote hain:

CPU (Central Processing Unit): Isko computer ka dimaag kehte hain, yeh saara kaam control karta hai.
ALU (Arithmetic Logic Unit): Calculation aur logic wale kaam karta hai. Example: Addition, Subtraction.
CU (Control Unit): Yeh sare parts ko samjhata hai ki kaam kaise karna hai.
MU (Memory Unit - RAM): Computer ki temporary memory hoti hai, jo kaam karte waqt data store karta hai. Computer band karne ke baad yeh memory clear ho jati hai.


3. Output Devices (Result dikhane wale tools):
Input aur processing ke baad computer jo result deta hai, use output devices ke through dikhate hain:

Monitor: Yeh ek screen hoti hai jo result ya images dikhati hai.
Printer: Yeh result ko paper pe print karta hai.
Types:
Laser Printer: Jaldi aur saaf print deta hai.
Inkjet Printer: Ink se color aur black print karta hai.
Dot Matrix Printer: Purane type ka printer jo points bana ke print karta hai.
Plotter: Badi images ya design (like maps, blueprints) print karta hai.


4. Utility Software (Helpful Software):
Yeh software system ko smooth chalane aur repair karne me madad karte hain:

Programming Languages: Software banane ke liye use hota hai. Examples:
C
C++
Python
Java


5. Virus (Computer ke liye khatarnak programs):
Kya hai: Virus ek program hota hai jo computer ko damage kar sakta hai.
Types:
Trojan: Hidden programs jo system ko harm karte hain.
Worm: Jaldi-jaldi computer ke files ko copy karke storage bhar deta hai.
Ransomware: Aapke files ko lock kar deta hai aur paise maangta hai unlock karne ke liye.
Effects:
Computer slow ho jata hai.
Files delete ho sakti hain.
Personal data chori ho sakta hai.
Protection: Antivirus software ka use karein jaise QuickHeal, Norton, McAfee.


6. Operating System (OS) aur uske Functions:
Operating System ek software hai jo computer ko chalata hai aur user ko system use karne me madad karta hai.

Functions (Kaam):
Process Management: Programs ko run karta hai.
Memory Management: RAM ka use karta hai aur data ko store karta hai.
File Handling: Files ko open, save, aur delete karne ka kaam karta hai.
Input/Output Management: Devices jaise mouse, keyboard, printer ko manage karta hai.
User Interface:
GUI (Graphical User Interface): Icons aur windows ke through easy use karta hai.
CLI (Command Line Interface): Commands type karke computer ko chalata hai.

Q1. Define Digital Computer? [2 Marks]
Digital Computer ek aise computer ko kehte hain jo data ko 0 aur 1 (binary form) mein samajhta hai. Yeh high speed pe kaam karta hai aur calculations, data processing, aur decision making karta hai.

Q2. Write the Types of Computers? [2 Marks]
Computers ke 4 main types hain:

Super Computer: Bohot powerful aur high-speed wale (scientific research ke liye).
Mainframe Computer: Bade companies aur banks ke liye (bohot saare users ka data handle karta hai).
Mini Computer: Chote organizations ke liye (medium size tasks).
Micro Computer: Personal use ke liye (PCs, Laptops).


Q3. Define MICR and OCR? [2 Marks]
MICR (Magnetic Ink Character Recognition): Yeh bank cheques ke niche likhe numbers ko padhne ke liye use hota hai jo magnetic ink mein likhe hote hain.
OCR (Optical Character Recognition): Yeh printed ya handwritten text ko scan karke computer-readable text mein badalta hai.


Q4. Difference Between Primary and Secondary Memory? [2 Marks]
Primary Memory	Secondary Memory
Temporary memory hoti hai (RAM).	Permanent memory hoti hai (HDD/SSD).
Fast aur directly CPU ke saath connected hoti hai.	Slow aur CPU ko indirectly access karti hai.
Data power off hone pe delete ho jata hai.	Data save rehta hai even after power off.


Q5. What is Language Translator? [2 Marks]
Language Translator ek software hai jo high-level programming languages (jaise C, Java) ko machine language (binary code) mein convert karta hai taaki computer use samajh sake.
Types:

Compiler
Interpreter
Assembler


Q6. What is Output Device? [2 Marks]
Output Device wo hota hai jo computer ke processed data ko user ko dikhata hai ya print karta hai. Examples:

Monitor: Screen pe results dikhata hai.
Printer: Paper pe results print karta hai.


Q7. Write the Components of Computer? [2 Marks]
Computer ke 4 main components hote hain:

Input Devices: Data input ke liye (Keyboard, Mouse).
Processing Unit: Data process karne ke liye (CPU).
Memory: Data store karne ke liye (RAM, Hard Disk).
Output Devices: Result dikhane ke liye (Monitor, Printer).

Fourth Generation (1971-Present): Microprocessors ka use hua.
